# 環境ファイルの使い方

## イメージをビルドする
主に、 **CI上** で使用する。
アプリケーションの実行はされず、イメージのビルドのみ行われる。
```
skaffold build
```

## 依存関係のみを立ち上げる
主に、 **e2eテスト** で使用する。
アプリケーション自体はk8sで実行せず、モックサーバーをローカルk8s上に立てる。
```
skaffold dev --port-forward
```

## アプリケーションのみを実行する
ローカルk8sでアプリケーションを起動するときに使用する。
依存関係はたたず、アプリケーションのリクエスト先も本番と同様になる。
```
skaffold run --port-forward
```
